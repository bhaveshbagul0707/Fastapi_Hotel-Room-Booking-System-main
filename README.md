# 🏨 Hotel Room Booking System (FastAPI)

This project is a **FastAPI-based backend application** developed as part of the **Feb Internship 2026 (Innomatics Research Labs)**.

It simulates a real-world hotel booking system with features like room management, booking system, cart workflow, filtering, sorting, and pagination.

---

## 🚀 Features Implemented

### 🔹 Day 1 – Basic APIs
- Home route
- Get all rooms
- Get room by ID
- Room summary
- Get all bookings

### 🔹 Day 2 – Validation
- Pydantic models
- Input validation (guest name, nights, phone, etc.)

### 🔹 Day 3 – Helper Functions
- Room search helper
- Cost calculation with:
  - Meal plans
  - Early checkout discount

### 🔹 Day 4 – CRUD Operations
- Add room
- Update room
- Delete room

### 🔹 Day 5 – Workflow (Cart System)
- Add room to cart
- View cart
- Remove from cart
- Checkout booking

### 🔹 Day 6 – Advanced Features
- Search rooms
- Filter rooms
- Sort rooms
- Pagination
- Combined browsing (search + pagination)
- Search bookings
